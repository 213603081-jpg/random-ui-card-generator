# Random UI Card Generator

一个使用纯 HTML、CSS 和 JavaScript 制作的随机 UI 设计灵感卡片生成器。

页面会随机展示不同的设计关键词、推荐配色、适用场景和风格描述，并根据当前灵感同步切换页面主题色。配色区域会显示对应色块和 HEX 色值，方便了解每组方案使用了哪些颜色。

## 文件结构

```text
random-ui-card-generator/
├── index.html
├── style.css
├── script.js
├── README.md
└── .gitignore
```

## 本地打开

这个项目不需要安装依赖，也不需要后端服务。

直接双击 `index.html`，或在浏览器中打开：

```text
index.html
```

## 部署

可以部署到任何静态网站托管平台，例如 GitHub Pages、Netlify 或 Vercel。

GitHub Pages 部署方式：

1. 将整个 `random-ui-card-generator` 文件夹上传到 GitHub 仓库。
2. 在仓库设置中打开 Pages。
3. Source 选择 `Deploy from a branch`。
4. Branch 选择 `main`，目录选择 `/root`。
5. 保存后等待 GitHub Pages 生成访问链接。
